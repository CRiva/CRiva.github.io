History Object

Python

An Object implementation that tracks added and changed elements.

>>> from history_object import History
>>> 
>>> @History()
>>> class T():
>>>     def __init__(self, x):
>>>        self.x = x
>>>
>>> test = T("Hello World")
>>> test.history['x']  # ["Hello World"]
>>> test.x = "Goodbye World"
>>> test.history['x']  # ["Hello World", "Goodbye World"]
